# Greymind Generator for Yo
Greymind.com generator for popular web frameworks and such.

## Usage
* `npm install -g generator-greymind`
* `yo greymind <AppName>`

## Generators
* `ngcontroller <ControllerName>`
* `ngview <ControllerName>`
